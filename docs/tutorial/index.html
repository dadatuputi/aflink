Browser Search Tutorial

Follow these instructions to search this site directly from your browser address bar.

Firefox

  1. Visit https://aflink.us
  2. Right-click on the address bar and select 'Add "aflink"'
  3. In your address bar, type 'aflink' then tab (⇥) to engage the search engine

Chrome

  1. Visit https://aflink.us
  2. Right-click on the address bar and select 'Manage search engines and Site Search'
  3. Scroll to the bottom, under 'Inactive shortcuts' and click 'Activate' next to 'aflink'
  4. In your address bar, type 'aflink' then tab (⇥) to engage the search engine

Edge

  1. Visit https://aflink.us
  2. In your address bar, type 'aflink' then tab (⇥) to engage the search engine
